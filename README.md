# 依赖

```Python>=3``` , ```requests```

# 用法

```
用法：
    python3 xdncovup.py [参数]
参数：
    -h,--help                   输出帮助信息
    -u,--username <学号>        指定学号
    -p,--password <密码>        指定密码
    -l,--location <上报地址>    指定上报地址（格式：某国某省某市某县/区）
    -d,--debug                  进入调试模式
```
